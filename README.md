# Nordic Snow Canvas

Minimal, elegant snow animation for a quiet Nordic night — pure HTML/CSS/JS canvas.

## Live Demo
Open the GitHub Pages URL shown in Settings → Pages.

## Features
- Subtle depth (size/opacity/speed linked)
- Gentle wind + sway
- Responsive (ResizeObserver)
- prefers-reduced-motion support

## Usage
Open `index.html` in your browser.
