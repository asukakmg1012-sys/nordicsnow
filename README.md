# Nordic Snow Canvas

Minimal, elegant snowfall for a quiet Nordic night — built with pure HTML/CSS/JS (Canvas).

## Live Demo
- https://asukakmg1012-sys.github.io/nordicsnow/

## Features
- Refined snowfall with subtle depth (size/opacity/speed linked)
- Gentle wind + organic sway (kept intentionally minimal)
- Responsive canvas (ResizeObserver + DPR support)
- `prefers-reduced-motion` support

## Getting Started
1. Download or clone this repository
2. Open `index.html` in your browser

## Customize
Edit these values in `index.html`:

- `CONFIG.intensity` (snow density): recommended `0.8–1.8`
- `CONFIG.wind` (horizontal drift): recommended `0.00–0.20`
- `CONFIG.dprMax` (render quality vs performance): `1` for low power devices, `2` for crisp visuals

## Deploy (GitHub Pages)
1. Repository → Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` / Folder: `/ (root)`
4. Open the published URL
